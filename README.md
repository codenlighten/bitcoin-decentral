## üöÄ QUICK START

### **Build Requirements**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install -y build-essential cmake git
sudo apt install -y libboost-all-dev libssl-dev
sudo apt install -y libdb++-dev libevent-dev
```

### **Build Bitcoin Decentral**
```bash
git clone https://github.com/codenlighten/bitcoin-decentral.git
cd bitcoin-decentral
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
```

### **Launch Network**
```bash
# Start Bitcoin Decentral daemon
./bitcoindecentrald -daemon

# Check status
./bitcoindecentral-cli getblockchaininfo

# Generate blocks (testnet)
./bitcoindecentral-cli generate 101
```
